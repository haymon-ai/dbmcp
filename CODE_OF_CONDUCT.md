# Code of Conduct

This repository exists for technical collaboration.

## Rules

- Be respectful and professional
- Keep discussions relevant to the project
- Critique code and ideas, not people
- Avoid spam, trolling, hostility, and repetitive derailment
- Respect maintainer decisions

## Enforcement

Maintainers may remove content, lock threads, or restrict participation if behavior becomes disruptive or unproductive.

## Contact

If you see misbehavior, ignore the person(s) and speak with a moderator or administrator.
